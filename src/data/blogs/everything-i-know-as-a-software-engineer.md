Back in 2019, I joined the Computer Engineer program in University of Engineering and Technology. I was young and dumb then, thinking that the major would be similar to IT stuffs, like creating some triple A games, recreate million dollar companies like Facebook, Google,... but then reality hit me. The major was all about designing chips, embedded programs, and making them work together.

It was too hard for me that I couldn't keep up. So I decided to learn something that I've dreaming of, that is to do Software Engineer.

## The First Steps

I took 6 - 8 months to learn the very basics of programming. Those free websites on the internet help a lot on my journey. Shout out to [f8](https://f8.edu.vn/) and [w3school](https://www.w3schools.com/) for making my dream come true.

Here's what I learned in that period:

- Programming with Javascript (variable, loop, async await, scope, if else statement,...)
- Basics about HTML (Hypertext Markup Language) - First step to web development
- Styling with CSS (Cascading Style Sheets) - I've learned to center a div
- Managing codebase and version control with Git
- With the 3 core concepts of web development, I've created [my first website](https://namnv2001.github.io/landing-coffee/)
- ReactJS - I chose React over Vue since it was more popular and had greater community support.
- React Native - I've created [a music player](https://github.com/namnv2001/nottify-react-native) for my Mobile Programming course at school.

## Internship (2022 - 2023)

I've landed my first internship interview ever at [Teko Solution](https://career.teko.vn/), onboard at April 28th, 2022. Here I've met amazing people and got to learn a lot more:

- Agile methodology at work
- Type safety with Typescript
- CI/CD for a smooth development experience
- A little bit about Goland, enough to read, understand, and able to edit the code to match the ticket's requirement
- Noticed about AI (Artificial Intelligence) and tried using Github Copilot
- Communicating with colleagues - Yeah, it's not tech-related but trust me, it's one of the most important things you gotta learn
- Writing unit tests with Jest and automation testing

## Official Staff (2023 - now)

After graduated, I've became an official staff at July 10th, 2023. Got moved to a new team, I've been the team's FE PIC since then. With a new role, I've learned many new things that still values till this day:

- Release process/ version control on production environment
- Monitoring and debugging API logs with Grafana, Elastic search (Log tracing, create dashboards,...)
- Setup and monitor Javascript error using Sentry
- Created and published my first SDK to NPM (Node Package Manager), called [teko-file-history](https://www.npmjs.com/package/teko-file-history/)
- Applying more AI into my workflow but won't rely on it (Researching, Coding support, Writing testcases,...)
- Time management - Since I had to handle 3 - 4 projects at a time
- Mentoring skills - I've got 2 interns under my management
- Dive a bit deeper into Goland, being able to implement and maintain simple APIs
- Writing and best practices for TDD (Technical Design Document)

On November 2025, I switched to consumer facing projects, facing new challenges:

- Optimizing Core Web Vitals and web performance
- How to do SEO (Search Engine Optimization) - basic things like sitemap, schema, tag management,...
